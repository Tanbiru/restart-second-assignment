1-What is JSX, and why is it used?
Ans: JSX  is a syntax that lets you write HTML-like code inside JavaScript in React.It is used to make UI code easier to read and write.

2- What is the difference between State and Props?
Ans: State: (i)Data that can change inside a component ,(ii)Managed inside the component
     Props : (i)Data passed from parent to child , (ii) Read-only (cannot change)

3- What is the useState hook, and how does it work?
Ans: useState is a React hook used to store and update state in a component.

4- How can you share state between components in React?
Ans: You can share state by lifting the state up to a parent component and passing it to child components using props.

5- How is event handling done in React?
ans: Event handling in React is done using camelCase event names and passing a function.
